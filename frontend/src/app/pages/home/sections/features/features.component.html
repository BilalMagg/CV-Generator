About the platform

What is CV Generator?

CV Generator is a microservices-based platform developed by a team of full-stack engineers. It leverages AI agents, vector search, and event-driven architecture to simplify the job application process.

🤖

AI Pipeline

A 5-agent LangGraph workflow extracts, matches, fills, optimizes, and delivers CVs automatically.

🔍

Semantic Matching

pgvector embeddings compare your profile against job descriptions to find the best fit.

📊

Application Tracking

Track every application through a full Kanban pipeline with Kafka-powered event updates.

⚡

ATS Optimization

Groq LLM rewrites CV content to maximize compatibility with applicant tracking systems.

📦

Auto Delivery

Completed CVs are stored in MinIO and delivered via email — no manual export needed.

🔐

Secure Architecture

Keycloak OIDC authentication with an API Gateway securing every microservice endpoint.
